# LaFesta

## 프로젝트 소개

라코스테를 모티브로 하여 진행한 라페스타 프로젝트에서 Main과 Nav/Footer를 반응형으로 제작하였습니다.

## 실행방법

`npm run start`

## 적용 기술

- React.js(Hooks)
- React-router-dom
- SCSS

## 구현 기능

- 브라우저 사이즈에 따라서 유동적인 메인 캐러셀 로직 구현.
- MockData를 활용한 메인 콘텐츠 내용 구성.
- 웹 퍼스트 방식을 사용한 Main, Nav 구현
- 모바일 퍼스트 적용한 Footer 구현.
- 버튼 클릭시 모달창 구현.
